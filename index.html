Ponystrap

Simple HTML/CSS/JS components for adding Django ponies to your websites.

Django Pony

First, add this HTML to your web page:

<div class="djangopony"></div>

Second, add this CSS to wherever you put your web page's custom CSS styles. Change the coordinates to be where you want your pony to be:

.djangopony{top:110px;left:700px;}

Third, add this JS to the bottom of your web page to make the pony's wings flutter:

$('.djangopony').sprite({fps: 12, no_of_frames: 2}).activeOnClick().active();

Installation

Copy ponystrap.css over to your project and link to it:

<link href="css/ponystrap.css" rel="stylesheet">
