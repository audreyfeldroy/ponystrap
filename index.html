Ponystrap

Simple HTML/CSS/JS components for adding Django ponies to your websites.

A Django pony

First, add this HTML to your web page. Replace 'yourpony' with the name of your pony:

<div class="djangopony" id="yourpony"></div>

Second, add this CSS to wherever you put your web page's custom CSS styles. Change the coordinates to be where you want your pony to be:

#yourpony{top:110px;left:700px;}

Third, if you want a static pony, add this JS to the bottom of your web page to make the pony's wings flutter:

$('#yourpony').sprite({fps: 12, no_of_frames: 2});

If you want the pony to fly to mouse clicks, use this JS instead:

$('#yourpony').sprite({fps: 12, no_of_frames: 2}).activeOnClick().active();
$('body').flyToTap();

Installation

Copy ponystrap.css over to your project and link to it:

<link href="css/ponystrap.css" rel="stylesheet">

Also copy over pony.png to your project's image directory.

Dependencies
